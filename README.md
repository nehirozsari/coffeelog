# ☕ Coffee Log

Coffee Log is a modern Web application built with **React.js** that allows coffee enthusiasts to document their daily brewing journeys. Users can log details about different coffee beans, record specific brewing methods, rate their experiences, and keep detailed tasting notes to refine their palate over time.

The application is fully responsive and has been successfully deployed to production.

🚀 **[Live Demo Link](https://nehirozsari.github.io/coffeelog/)**

---

## 🛠️ Tech Stack & Features

### Core Technologies
* **Frontend Library:** React.js (Hooks, Functional Components)
* **Styling:** Custom CSS3 (Fully Responsive Design)
* **Build Tool:** Create React App (CRA)
* **Package Manager:** npm

### Key Features
* **Interactive Logger:** Create, read, update, and delete daily coffee logs.
* **Brewing Customization:** Track variables like bean origin, roast level, grind size, and water temperature.
* **Tasting Notes:** Log flavor profiles (fruity, nutty, chocolatey, etc.) and personal ratings.
* **Production Ready:** Optimized production build deployed seamlessly via automated workflows.

## 📦 Installation and Local Setup

Follow these steps to set up and run the project locally on your development machine:

1. **Clone the Repository**
   ```bash
   git clone [https://github.com/nehir/CoffeeLog.git](https://github.com/nehir/CoffeeLog.git)
   cd "Coffee Log"

   **Install Dependencies**
   npm install
   **Start the Development Server**
   npm start
